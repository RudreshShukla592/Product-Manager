# Product Manager

A CRUD-based product management application built using **HTML**, **CSS**, and **JavaScript**, with **Local Storage** integration for persistent data management.

## ✨ Features

* Add New Products
* View Product Listings
* Update Existing Products
* Delete Products
* Data Persistence with Local Storage
* Dynamic DOM Manipulation
* Responsive User Interface

* CRUD Operations

* Form Validation
* JavaScript Data Structures
* State Management without Frameworks

## 🎯 Purpose

This project was created to strengthen my JavaScript fundamentals by building a real-world CRUD application. It focuses on managing product data, handling user interactions, and persisting information using the browser's Local Storage.

## 🚀 Key Learning Outcomes

* Creating and managing dynamic UI components
* Working with browser storage APIs
* Implementing Create, Read, Update, and Delete functionality
* Organizing JavaScript code for maintainability
* Building practical frontend applications without external libraries
